#                    Кнопки и их функции на телефонах Yealink

| **Тип кнопки**                         | **Что делает**                                | **Эффект при нажатии**                          | **Лампочка / индикатор**        |
|----------------------------------------|----------------------------------------------|------------------------------------------------|---------------------------------|
| **Быстрый набор (Speed Dial)**         | Одно нажатие → сразу звонок                  | Набирается номер                               | нет                             |
| **BLF**                                | Показывает статус сотрудника                 | Звонок / перехват                              | свободен, занят                 |
| **Аккаунт (Account)**                  | Переключение активного аккаунта телефона     | Телефон меняет активный SIP аккаунт            | нет                             |
| **Парковка (Call Park)**               | Кладёт звонок на парковку                    | Звонок ставится на парковку                    | нет                             |
| **Перезвонить (Redial)**               | Набрать последний вызов                      | Номер набирается автоматически                 | нет                             |
| **Конференция с (Conference)**         | Добавление участника к разговору             | Создаёт конференцию                            | нет                             |
| **Использование DND запрещено**        | Игнорирует DND на телефоне                   | Нет реакции на DND                             | нет                             |
| **DTMF**                               | Отправка DTMF тонов                          | Вызов или управление АТС                       | нет                             |
| **Контакты**                           | Открывает адресную книгу телефона           | Можно выбрать номер                            | нет                             |
| **Перед. (Forward)**                   | Перевод звонка на другой номер              | Звонок переводится                             | нет                             |
| **Гр. прослушивание (Group Listening)**| Слушать разговор в группе                   | Слушаете чужой звонок                          | нет                             |
| **Гр. захват (Group Pickup)**          | Перехват звонка группы                       | Принимаете звонок другого телефона             | нет                             |
| **Баланс**                             | Проверка баланса (VoIP/телефония)           | Отправка запроса на сервер                     | нет                             |
| **Удержание (Hold)**                   | Поставить звонок на удержание               | Звонок удержан                                 | мигает удержание                |
| **Личное удержание (Private Hold)**    | Удержание только на этом телефоне           | Звонок удержан                                 | мигает удержание                |
| **Лок. группа (Local Group)**          | Управление внутренней группой               | Звонки группы / BLF                            | зависит от конфигурации         |
| **Группа оповещения (Paging Group)**  | Одновременное объявление на группу          | Звук на всех телефонах группы                  | нет                             |
| **Список пейджинга (Paging List)**    | Голосовое объявление                        | Звонит на телефоны списка                      | нет                             |
| **Listening List**                     | Прослушивание линий                         | Можно слушать разговоры                        | нет                             |
| **Pickup**                             | Перехват чужого звонка                      | Берёте звонок                                  | нет                             |
| **Префикс (Prefix)**                   | Добавить код перед набором                  | Используется для маршрутизации звонка          | нет                             |
| **Личное удержание**                   | Поставить звонок на удержание только на этом телефоне | Звонок удержан                            | мигает удержание                |
| **Запись голоса (Call Recording)**     | Начало записи звонка                        | Звонок записывается                            | нет                             |
| **SMS**                                | Отправка SMS                                | Открывает окно отправки                        | нет                             |
| **Трансфер (Transfer)**                | Перевод звонка                              | Звонок переводится                             | нет                             |
| **Голосовая почта (Voicemail)**        | Доступ к voicemail                          | Слушаете сообщения                             | нет                             |
| **XML-браузер / URL / URL-запись**     | Веб-сервисы / команды                       | Открывает страницу или отправляет команду      | нет                             |
| **Удалённая книга (Remote Phonebook)** | Открывает удалённую телефонную книгу        | Можно выбрать номер                            | нет                             |
| **Ztouch**                             | Автонастройка телефона                      | Обновление через provisioning                  | нет                             |
| **Блокировка клавиатуры (Key Lock)**   | Блокировка телефона                         | Нажатия игнорируются                           | заблокировано                   |
| **Hot Desking**                        | Вход в чужой аккаунт телефона               | Телефон работает под вашим профилем            | нет                             |
| **Интерком (Intercom)**                | Мгновенная громкая связь                    | Громкая связь без трубки                       | нет                             |
| **BLF список (BLF List)**              | Отображение нескольких BLF                 | Показывает статус всех пользователей списка    | свободен, занят                 |
| **Вернуть с парковки (Unpark)**       | Забрать звонок с парковки                   | Звонок появляется на телефоне                  | нет                             |
| **ACD**                                | Работа с очередью (колл-центр)              | Вход/выход из очереди                          | нет                             |
